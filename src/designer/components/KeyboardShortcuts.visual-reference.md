## Usage

### Opening the Modal

The modal can be opened by:
1. Pressing `F1` anywhere in the Designer Mode
2. Clicking the keyboard shortcuts button in the header (ğŸ¹ icon)

### Closing the Modal

The modal can be closed by:
1. Pressing `Escape`
2. Clicking the Ã— button in the header
3. Clicking outside the modal (on the backdrop)
4. Clicking the "Got it" button
